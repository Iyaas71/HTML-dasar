Apa itu HTML ?

HTML itu adalah sebuah bahasa yang menggunakan markup atau penanda untuk membuathalaman web. Penanda atau markup ini, nanti akan kita sebut dengan tag. Penanda atau markup ini , nanti akan kita sebut dengan HTML berperan untuk menentukan struktur konten dan tampilan dari sebuah web Kalau kita ibaratkan nih..HTML itu seperti batu bata untuk membangun rumah. Batu bata ini dapat disusun, hingga menjadi fondasi dasar. HTML memiliki kelebihan seperti memiliki sumber komunitas yang cukup besar dengan pengguna yang luas.Tidak hanya itu, kelebihan lainnya adalah HTML mudah untuk dipelajari oleh seorang Web Developer Pemula.kelebihan lainnya adalah HTML merupakan bahasa pemrogaman yang gratis atau open source
